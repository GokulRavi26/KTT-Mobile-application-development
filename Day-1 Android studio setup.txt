Step 1: Install the latest version of Android Studio.
Step 2: Install the required SDK packages and set the SDK path if needed.
Step 3: Create a new project.
Step 4: Choose Kotlin as the programming language.
Step 5: Enter your app name.
Step 6: Click Create to generate the project.
Step 7: Open Device Manager and connect your mobile device using either USB or Wi-Fi debugging.
Step 8: On your phone, enable Developer Options and allow debugging access when prompted.
Step 9: For wireless debugging, make sure your phone and PC are connected to the same Wi-Fi network.
Step 10: Click the Run button to install and test your app on the connected device.
